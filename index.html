Overview

CASA-Lite analyzes fish sperm motility using computer vision techniques. It processes microscopy videos to track individual sperm cells, calculate motility parameters, and generate comprehensive reports.

Installation

Clone the repository and install dependencies:

git clone https://github.com/temabef/CASA-Lite.git
cd CASA-Lite
pip install -r requirements.txt

Usage

Start the web interface:

python run.py --web

Then open your browser and navigate to http://localhost:5000

Or process a video file directly:

python run.py --video path/to/video.mp4

Key Features

  • Video Processing: Extract and process frames from microscopy videos
  • Sperm Tracking: Advanced algorithm to track individual sperm cells across frames
  • Motility Analysis: Calculate VCL, VSL, VAP, LIN, WOB, BCF and other parameters
  • Visualization: Generate trajectory plots and velocity distribution histograms
  • Reporting: Create comprehensive HTML reports with all analysis data
  • Web Interface: Easy-to-use interface for uploading and analyzing videos

For Researchers

CASA-Lite is designed as an affordable alternative to expensive commercial CASA systems. It's particularly useful for:

  • Aquaculture research facilities with limited budgets
  • Academic research in reproductive biology
  • Educational purposes in fisheries and aquaculture programs
  • Field research where portable, software-based solutions are needed
